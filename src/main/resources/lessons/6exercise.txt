Создайте класс с именем Device. Он должен содержать метод setCode, принимающий на вход целое число и строку со значением. Класс должен содержать метод getInfo, позволяющий по известному коду выдать строку с информацией. И класс должен иметь еще один метод, позволяющий получить список кодов - getCodes().
